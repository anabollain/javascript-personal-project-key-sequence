# JS Key Sequence

## Description

This exercise consists on executing a function when the user types a certain word. 

In this application, whenever the user types 'Frida', her picture will appear on the screen.

## Javascript Used

* document.querySelector()
* splice()
* join()
* toLowerCase()
* includes()
* background property
* innerHTML
* keyup event
